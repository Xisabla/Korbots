## How to contribute

0 - Pull !
```bash
git pull
```
1 - Code your changes
Do your stuff :)

2 - Commit your changes
```bash
git add . # or git add myfile.ext myfolder/
git commit -m "my commit description"
```
3 - Push your changes
```bash
git pull
```
